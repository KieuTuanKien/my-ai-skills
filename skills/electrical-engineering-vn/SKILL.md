# Kỹ sư Điện - Hệ thống Điện Việt Nam

## Tiêu chuẩn áp dụng
| Lĩnh vực | Tiêu chuẩn |
|----------|------------|
| TBA | IEC 62271-200 (tủ MV), IEC 62271-100 (MC), IEC 61869 (CT/VT) |
| MBA | IEC 60076 |
| Cáp | IEC 60502, IEC 60228, TCCS 17:2021/EVN |
| Rơle | IEC 60255, IEC 61850 |
| SCADA | IEC 60870-5-104, IEC 61850, Modbus |
| Tiếp địa | TCVN 9358, IEEE 80 |
| PCCC | QCVN 02:2020/BCA, TCVN 3890:2023 |

## Quy định EVN/EVNHN/EVNNPC
- QĐ 171/HĐTV: Tiêu chuẩn RMU 22kV/35kV
- QĐ 2838/EVNHN: Tủ hợp bộ AIS 22kV/35kV
- QĐ 5411/EVNHN: Tủ điều khiển bảo vệ nhị thứ TBA 110-220kV
- QĐ 55/ĐTĐL: SCADA quản lý vận hành
- TT 05/2025/TT-BCT: Truyền tải, phân phối, đo đếm
- TT 02/2025/TT-BCT: Kiểm định an toàn kỹ thuật
- STL members: ASTA (Intertek), KEMA (DNV), XIHARI, CESI, KERI, CPRI

## Phạm vi
- Thiết kế sơ đồ nhất thứ (SLD), nhị thứ
- Tính toán ngắn mạch, chọn thiết bị
- Cấu hình rơle bảo vệ, logic liên động
- Tích hợp SCADA: danh sách tín hiệu, file SCL/SCD
- Cáp trung thế: lựa chọn tiết diện, sụt áp, đầu cáp
- Tiếp địa: tính toán điện trở, lưới tiếp địa, chống sét
